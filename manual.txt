Установка фреймворка и компонентов:
    pip install flask
    pip install flask_sqlalchemy

Установка бд:
    В режиме python:
        >>> from app import app, db
        >>> app.app_context().push()
        >>> db.create_all()
        бд появляется в папке instance под указанным в главном файле названии
        from app - имя главного файла приложения, app db - переменные приложения flask и базы данных внутри главного файла

Шаблоны в папке templates:
    base.html - основной шаблон, в котором находятся все повторяющиеся элементы сайта (шапка, футер, подключение стилей и т.п.)
    index.html - шаблон главной страницы, на которой выводятся все текущие задачи
    about.html - шаблон страницы о сайте
    tasks.html - все когда-либо созданные задачи
    tasks_completed.html - закрытые задачи
    tasks_overdue.html - просроченные, незакрытые задачи
    task_detail.html - страница с конкретной задачей, с возможностью изменить её
    update_task.html - страница для изменения задачи (get и post запросы)
    create_task.html - страница для создания задачи (get и post запросы)

Папка static:
    статические файлы для использования на сайте. Внутри хранятся файлы стилей CSS или картинки. В проекте используются
    только файлы стилей - main.css

Папка instance:
    хранит базу данных проекта. Создаётся автоматически - установка выше. В нашем случае task_manager.db

Папка venv:
    виртуальное окружение проекта. Все необходимые фреймворки внутри.

Файл app.py:
    главный файл проекта, где хранится вся логика бэкенда. Модель Task для хранения задач и функции для обработки страниц.

Файловая структура:
    \instance
        \task_manager.db - база данных
    \static
        \css
            \main.css - файл стилей
    \templates - папка с шаблонами, подробнее выше
        \ base.html
        \index.html
        \about.html
        \tasks.html
        \tasks_completed.html
        \tasks_overdue.html
        \task_detail.html
        \update_task.html
        \create_task.html
    \venv - виртуальное окружение, формируется и собирается автоматически в pycharm
        \...
    \app.py - главный файл приложения
